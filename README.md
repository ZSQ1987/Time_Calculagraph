# 多任务倒计时工具

一款功能强大、界面美观的多任务倒计时工具，支持倒计时任务和定时提醒，带系统托盘和悬浮窗功能，让时间管理变得更加高效。

## 🎯 功能特性

### 核心功能
- **多任务管理**：同时支持多个倒计时任务和定时提醒
- **可视化界面**：美观的GUI界面，支持任务分类显示
- **系统托盘**：最小化到系统托盘，不占用桌面空间
- **悬浮窗**：显示任务列表和目标时间，随时查看任务状态
- **任务存档**：退出后自动保存任务，下次启动时恢复
- **智能提醒**：任务到期时弹出提醒窗口，支持不同类型任务的不同提醒颜色

### 特色功能
- **目标时间显示**：在悬浮窗中显示每个任务的目标北京时间
- **任务复制**：支持复制现有任务，快速创建类似任务
- **时间计算**：智能计算任务剩余时间，确保目标时间准确无误
- **界面优化**：响应式布局，美观的颜色区分，良好的用户体验

## 📥 安装方法

### 方法一：直接运行（推荐）
1. 下载 `dist/time.exe` 文件
2. 双击运行即可使用
3. 无需安装Python环境

### 方法二：源码运行
1. 确保已安装Python 3.7+
2. 安装依赖：
   ```bash
   pip install pystray Pillow psutil
   ```
3. 运行：
   ```bash
   python time.py
   ```

## 🚀 使用指南

### 1. 添加倒计时任务
1. 在「倒计提醒」区域输入任务名称
2. 设置倒计时时长（时、分、秒）
3. 点击「添加任务」按钮
4. 任务将显示在「当前任务列表」中

### 2. 添加定时提醒
1. 在「定时提醒」区域输入提醒名称
2. 设置提醒日期和时间
3. 点击「添加提醒」按钮
4. 任务将显示在「当前任务列表」中

### 3. 管理任务
- **删除任务**：点击任务行中的「删除」按钮
- **复制任务**：点击任务行中的「复制」按钮，修改后创建新任务

### 4. 系统托盘操作
- **最小化**：点击窗口最小化按钮，程序将最小化到系统托盘
- **显示窗口**：点击托盘图标，选择「显示窗口」
- **显示悬浮窗**：点击托盘图标，选择「显示悬浮窗」
- **退出程序**：点击托盘图标，选择「退出」

### 5. 悬浮窗使用
- **查看任务**：悬浮窗显示所有当前运行的任务
- **目标时间**：每个任务下方显示目标北京时间
- **隐藏悬浮窗**：点击悬浮窗中的「隐藏」按钮

## ❓ 常见问题

### Q: 任务数量有限制吗？
A: 为了保证界面美观和性能，最多支持同时运行10个任务。

### Q: 任务会在电脑重启后保存吗？
A: 是的，程序会在退出时自动保存任务，下次启动时恢复。

### Q: 可以修改任务的目标时间吗？
A: 可以通过复制任务的方式修改目标时间，创建新任务。

### Q: 提醒窗口会一直显示吗？
A: 提醒窗口会在任务到期时弹出，需要手动点击「关闭」按钮关闭。
